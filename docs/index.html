Composer

Un administrador de dependencias para PHP

Composer es una herramienta para la gestión de dependencias en PHP. Le permite declarar las bibliotecas de las que depende su proyecto y las administrará (instalará/actualizará) por usted.

Descargar aquí

Composer Init

El comando composer init le pedirá de forma interactiva que complete los campos, mientras utiliza algunos valores predeterminados inteligentes para inicializar el proyecto.

Composer require

El comando composer require agrega nuevos paquetes al archivo composer.json desde el directorio actual. Si no existe ningún archivo, se creará uno sobre la marcha.

Composer install

El comando composer install lee el archivo composer.json del directorio actual, resuelve las dependencias y las instala en la carpeta vendor.

Composer remove

El comando composer remove elimina paquetes del archivo composer.json del directorio actual.
Después de eliminar los requisitos, se desinstalarán los requisitos modificados.

PHP GoogleTranslate Free

Traduce tu texto de un idioma a otro con esta increible SDK

SDK adaptada para php hecha por Statickidz, la cual nos permite traducir un texto desde un idioma origen a un idioma destino!

Descargue aquí

Instalación

Instala este paquete vía Composer con el siguiente código:

composer require statickidz/php-google-translate-free

O edita tu composer.json de tu projecto para requerir
statickidz/php-google-translate-free y luego ejecute composer install

"require": { "statickidz/php-google-translate-free": "^1.2.1"}

Uso

La integración se realiza mediante:

require_once ('vendor/autoload.php');

use \Statickidz\GoogleTranslate;
$source = 'es';
$target = 'en';
$text = 'buenos días';
$trans = new GoogleTranslate();
$result = $trans->translate($source, $target, $text);

// Good morning
echo $result;

Variable $source string que almacena el idioma origen al modo de la ISO 639-1, donde se puede encontrar como se abrevia cada lenguaje. Lo mismo con $target en el cual se escoge el idioma destino. En la variable $text se almacenará el texto a traducir.

Con new GoogleTranslate creamos el objeto que permitirá llamar a la funcion traducir, la cual es invocada con
$trans->translate($source, $target, $text);
En la cual se introduce en orden los parametros, la variable con el idioma origen, el idioma destino, y finalizalmente el texto a traducir. Esta función devuelve el texto traducido.

Google Text-to-speech

Dale voz a tu texto

El módulo gTTS produce una voz muy natural, como la de un humano, mientras que algunas otras API proporcionan voz como la de un robot. Está disponible gratuitamente y es fácil de implementar, ya que simplemente convierte el texto especificado en audio, que podemos guardar como un archivo mp3. Tiene la capacidad de leer una gran cantidad de texto con la entonación adecuada, abreviaturas, decimales y más.

Ver fuente

¿Cómo generar el tts?

Para generar el audio, hay que utilizar file_get_contents() con la url de la API y los atributos necesarios para generar el audio. Por ejemplo: file_get_contents('https://translate.google.com/translate_tts?ie=UTF-8&client=gtx&q='.$txt.'&tl=en-IN'); generaría el audio con el texto guardado en la variable $txt en inglés de la India.

Atributos de la URL

Tiene algunos atributos de URL predefinidos que se aceptan en solicitudes GET, como:
q- El texto a convertir.
tl- Idioma de traducción, ver idiomas y voces compatibles.
ie- Formato de codificación de entrada, utilizar el UTF-8 predeterminado.


Mostrar el resultado

Para mostrar y poder reproducir el audio hay que agregar en PHP: $speech="<audio controls='controls' src='" . $audio . "></audio>"; echo $speech;
Donde $audio es lo obtenido al usar file_get_contents().

Problemas

En caso de tener problemas a la hora de reproducir el audio, se puede hacer lo siguiente:
$file = file_get_contents($url);
$audio = "data:audio/wav;base64," . base64_encode($file);
Esto permite crear una URL dinámica local sin necesidad de descargar el archivo para que se reproduzca.
