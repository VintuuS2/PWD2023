Composer

Un administrador de dependencias para PHP

Composer es una herramienta para la gestión de dependencias en PHP. Le permite declarar las bibliotecas de las que depende su proyecto y las administrará (instalará/actualizará) por usted.

Descargar aquí

Composer Init

El comando composer init le pedirá de forma interactiva que complete los campos, mientras utiliza algunos valores predeterminados inteligentes para inicializar el proyecto.

Composer require

El comando composer require agrega nuevos paquetes al archivo compositor.json desde el directorio actual. Si no existe ningún archivo, se creará uno sobre la marcha.

Composer install

El comando composer install lee el archivo compositor.json del directorio actual, resuelve las dependencias y las instala en la carpeta vendor.

Composer remove

El comando composer remove elimina paquetes del archivo compositor.json del directorio actual.
Después de eliminar los requisitos, se desinstalarán los requisitos modificados.

PHP GoogleTranslate Free

Traduce tu texto de un idioma a otro con esta increible SDK

SDK adaptada para php hecha por statickidz, la cual nos permite traducir un texto desde un idioma origen a un idioma destino!

Descargue aquí

Instalación

Instala este paquete vía Composer con el siguiente código:

composer require statickidz/php-google-translate-free

O edita tu composer.json de tu projecto para requerir
statickidz/php-google-translate-free y luego ejecute composer install

"require": { "statickidz/php-google-translate-free": "^1.2.1"}

Uso

La integración se realiza mediante:

require_once ('vendor/autoload.php');

use \Statickidz\GoogleTranslate;
$source = 'es';
$target = 'en';
$text = 'buenos días';
$trans = new GoogleTranslate();
$result = $trans->translate($source, $target, $text);

// Good morning
echo $result;

Variable $source string que almacena el idioma origen al modo de la ISO 639-1, donde se puede encontrar como se abrevia cada lenguaje. Lo mismo con $target en el cual se escoge el idioma destino. En la variable $text se almacenará el texto a traducir.

Con new GoogleTranslate creamos el objeto que permitirá llamar a la funcion traducir, la cual es invocada con
$trans->translate($source, $target, $text);
En la cual se introduce en orden los parametros, la variable con el idioma origen, el idioma destino, y finalizalmente el texto a traducir. Esta función devuelve el texto traducido.
